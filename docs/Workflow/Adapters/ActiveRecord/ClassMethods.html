Overview

This module will automatically generate ActiveRecord scopes based on workflow states. The name of each generated scope will be something like with_<state_name>_state

Examples:

Article.with_pending_state # => ActiveRecord::Relation Payment.without_refunded_state # => ActiveRecord::Relation Example above just addswhere(:state_column_name => 'pending')or where.not(:state_column_name => 'pending')` to AR query and returns ActiveRecord::Relation.

def workflow_column(column_name = nil)
